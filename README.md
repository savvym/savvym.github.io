## Components
- [Hugo](https://gohugo.io)
- [LoveIt](https://github.com/dillonzq/LoveIt)

## How to use
Install Hugo
```bash
brew install hugo
```
Clone this repository
```bash
git clone https://github.com/savvym/savvym.github.io.git
```
Run hugo server
```bash
cd savvym.github.io.git 
git submodule init
git submodule update
hugo server -D
```
